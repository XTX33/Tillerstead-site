How to Use These Tokens

To change a token value:

  1. Open src/styles/tokens.css
  2. Find the token you want to modify (e.g., --color-primary)
  3. Update its value
  4. Save the file and refresh your browser

Example:

/* Change primary color from green to blue */
--color-primary: #3b82f6;  /* was #1ac87a */

All instances throughout the site will automatically update!
